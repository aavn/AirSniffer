Sniffer in a bottle version
- ESP8266 v3
- DHT11
- Nova SDS011

On-boarding instruction:
- Connect & disconnect 2 cables labelled 'Reset'
- On mobile or laptop: connect to wifi network 'Sniffer'
- Open web browser, go to: http://192.168.4.1
- Choose the wifi network that you want Sniffer to use
- Fill wifi password & Sniffer information like in the email you received. Press 'Connect' button
- Wait around 1 minute. Result: Green light - OK. Red light - Not OK
