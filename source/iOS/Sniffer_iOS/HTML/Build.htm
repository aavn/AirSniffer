How to build a Sniffer

1. Overview

Sniffer is a solution to address air pollutant. Sniffer will collect the following data:

  • Fine particles (PM2.5)
  • Coarse particles (PM10)
  • Humidity & Temperature
And send to AAVN cloud via wifi every 15 minutes. The collected data is available for the community at: https://crowdsniffing.org

4. Install IDE

We recommend use Arduino IDE (download from https://www.arduino.cc/) with ESP8266 plugin since we can re-use many sensor libraries from Arduino. You can follow the instruction from the link below to prepare IDE http://arduino.esp8266.com/stable/package_esp8266com_index.json

Needed libraries:

  1. Library for temperature and humidity sensor DHT11 or DHT22
  2. Json Library for Arduino
  3. Nova sensor library to measure dust data (PM2.5 and PM10) in the environment
  4. RestService library to send data to crowdsniffing.org network
  5. SnifferWifi to connect wifi and onboarding Sniffer to your wifi

You can get all the libraries here:​ https://github.com/aavn/AirSniffer/tree/master/source/libraries. Put them to folder \libraries of Arduino sketch directory. Restart Arduino IDE and start using all of libraries. You can find the sketch folder in File/Preference (Windows) or Arduino/Preferences… (Mac). If the \libraries folder does not exist, you can simply create it.

5. Get MAC Address

Normally, you can ask the seller the MAC address of the ESP8266 board. Otherwise, you can follow these steps to get the MAC address

  1. Connect the ESP8266 MCU to computer using USB
  2. Open Arduino IDE and create a new program (File->New), save it.
  3. Select menu Tools -> Board and select ‘NodeMCU 1.0 (ESP-12E Module) and port (the USB port connected to the MCU)
  4. Open Serial Monitor (by click the top-right button on the IDE)

  5. Paste the following code into the file has just been created

    #include <ESP8266WiFi.h>
    void setup(){
       Serial.begin(115200);
       delay(500);
       Serial.println();
       Serial.print("MAC: ");
       Serial.println(WiFi.macAddress());
    }
    void loop(){}

  6. Click ‘Upload’ button and get the Mac address from Serial Monitor. You will need the MAC address when register your sniffer to the network.
    Note: Make sure that you select the correct baud rate in the Serial Monitor screen (in the example it’s 115200 - the same in source code set `Serial.begin(​115200​);`)

6. Register Sniffer to ID

  1. Use MAC address to register: please email to the following address with the detail to get the Sniffer ID: aavn-iot-lab@axonactive.com ​

    • Title: Request for Sniffer Creation
    • Sniffer Name: name of your Sniffer
    • Address: Country, State, Province, City, Street, House number
    • GPS coordinate (Longitude and Latitude) (You can use tool like https://www.gps-coordinates.net​)
    • MAC Address of the MCU
  2. Within 1 working day, we will email you back the Sniffer’s ID that you will use to post your data to the network.
7. Build and load firmware

Get Sniffer source code and upload to hardware

You will get all necessary source code here: ​https://github.com/aavn/AirSniffer/tree/master/source/Sniffer

Upload code for ESP8266

Upload source code to ESP board: In Arduino IDE

  1. Choose: Tools/Board/Node MCU 1.0 (ESP-12E Module)
  2. Choose: Tools/Port and select correct USB port that your board plugin
  3. Press Upload
  4. Restart your board (press Reset button onboard)

9. Set up and run

Configure WiFi
When you need to update the Wifi information, please follow the steps:

  1. Press Reset button (You can easily use a chopstick to reach the button). Now Red led will blink.
  2. Connect your laptop or mobile, tablet to the ‘SNIFFER’ wifi.
  3. Open the browser, go to address: http://192.168.4.1​
  4. Choose the Wifi network that you want Sniffer to use. (NOTE: it must be a simple Wifi, with Name and with or without password)
  5. Type the wifi password, Sniffer ID (got from here), Coordinate values (as mentioned here) and the MAC Address (guidance can find here). Then press “Connect’’ button
  6. The message “Configuration stored. Restarting!” will show. Sniffer saves and uses configuration to send data to server.
  7. Wait around 1 minute for Sniffer to apply configuration and start sending data to server. You can check your data on https://crowdsniffing.org. If Red led blinks or you cannot see your sniffer on our network, re-check Configuration information. (Please keep in mind that if the Sniffer ID, GPS, MAC do not match with the information you registered with us, Sniffer cannot send data to our network)

Placement

We recommend using water bottle to take advantages of the waterproof ability. However, because we cut and re-install the bottle, so of course its stability is not as original. Hence, the best placement would be hanging in the balcony or rooftop. You should avoid extreme environment (Eg: direct expose to stormy weather, acidic …).
